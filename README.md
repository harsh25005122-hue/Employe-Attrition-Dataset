# Employee Attrition Prediction

## Project Overview
This project uses Machine Learning to predict employee attrition (whether an employee is likely to leave the company or stay). The model helps organizations identify employees at risk of leaving and supports better workforce management.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook

## Machine Learning Algorithm
- Random Forest Classifier

## Dataset
IBM HR Analytics Employee Attrition Dataset

## Features Used
- Age
- Department
- Education
- Gender
- Job Role
- Monthly Income
- Years At Company
- Job Satisfaction
- Work Life Balance
- Business Travel
- Environment Satisfaction
- And Other Employee Attributes

## Target Variable
- Attrition
  - Yes = Employee Leaves Company
  - No = Employee Stays in Company

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Label Encoding
4. Train-Test Split
5. Model Training
6. Prediction
7. Performance Evaluation

## Results
The Random Forest model successfully predicts employee attrition and helps identify factors that influence employee turnover.

## Conclusion
Employee attrition prediction can help companies improve retention strategies, reduce turnover costs, and make better HR decisions using data-driven insights.
